This tool is used to automatically fill and input your data into the Pennsylvania
  unemployment compensation's chat service.  Please download and install the ViolentMonkey
  extension for either chrome or firefox.

1. Download and change the AutoChat.js file to match your specific information
  1a. You should only be changing the constants: FIRST, LAST, EMAIL, PHONE and SUBJECT
  1b. Please see the list of subjects for your specific need
2. Install the ViolentMonkey extension for firefox or chrome
3. Create a new ViolentMonkey script and copy/paste your adjusted JavaScript File
    into the default editor and click "Save and Close"
4. Make sure your script is enabled under your ViolentMonkey dashboard
5. Go to https://www.uc.pa.gov/Chat/index.aspx and the script will start running automatically
